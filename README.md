# 報餐系統（Meal Order System）

## 簡介

企業內部報餐系統前端專案，提供自助餐、便當、套餐三種報餐方式，員工可依據每日菜單選擇餐點並送出報餐。

## 技術棧

| 項目       | 版本 / 說明                |
| ---------- | -------------------------- |
| Vue        | 3.5+                       |
| Vite       | 6.x                        |
| TypeScript | 5.7+                       |
| Pinia      | 2.x（狀態管理）            |
| Vue Router | 4.x（路由管理）            |
| Ant Design Vue | 4.2.6（UI 元件庫）     |
| SCSS       | 搭配 BEM 命名規範          |

## 快速開始

```bash
npm install
npm run dev
```

開發伺服器會啟動在 `http://localhost:5173`。

## 建置

```bash
npm run build
npm run preview
```
